Defect type quick guide
  • Vague wording → ambiguous_term
  • Missing responsible actor or system → missing_actor
  • Missing trigger or activation condition → missing_trigger
  • Missing observable expected outcome → missing_expected_outcome
  • Missing important constraint or boundary → missing_constraint
  • No clear pass/fail test → not_testable
  • Scope too broad or bundled behavior → overly_broad_requirement
  • Contradictory condition or result → inconsistent_condition
  • Requires unsupported outside assumption → unsupported_external_assumption

minor: small issue, main intent remains usable. major: likely to affect implementation or testing. critical: unusable or seriously contradictory.
